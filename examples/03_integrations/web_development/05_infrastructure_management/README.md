# Demo 05: Infrastructure Management

Demonstracja zarządzania infrastrukturą web application.

## Zawartość

- `demo.py` - Komendy zarządzania infrastrukturą
- `run.sh` - Skrypt uruchamiający demo

## Uruchomienie

```bash
./run.sh
```

## Co pokazuje ten przykład

- Klasa `InfraCommand` z komendami infrastruktury
- Scenariusze zarządzania:
  - Skalowanie usług
  - Monitorowanie statusu
  - Aktualizacja (rolling update)
  - Konfiguracja w locie
  - Backup i przywracanie
