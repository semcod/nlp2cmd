# Demo 02: Deployment Planning

Demonstracja planowania wdrożenia usług.

## Zawartość

- `demo.py` - Planowanie wdrożenia (DeploymentPlan)
- `run.sh` - Skrypt uruchamiający demo

## Uruchomienie

```bash
./run.sh
```

## Co pokazuje ten przykład

- Klasa `DeploymentPlan` z konfiguracją wielu usług
- Metoda `summary()` z podsumowaniem planu
- Zależności między usługami
- Konfiguracja sieci Docker
