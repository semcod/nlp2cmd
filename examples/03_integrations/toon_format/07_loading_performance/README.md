# Demo 07: Loading Performance

Porównanie wydajności ładowania TOON vs stary system.

## Zawartość

- `demo.py` - Benchmark wydajności
- `run.sh` - Skrypt uruchamiający demo

## Uruchomienie

```bash
./run.sh
```

## Co pokazuje ten przykład

- Benchmark starego systemu (wiele plików JSON)
- Benchmark TOON (pojedynczy plik)
- Porównanie czasów ładowania
- Obliczanie przyspieszenia (speedup)
- Wnioski z porównania
