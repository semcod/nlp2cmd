# Demo 13: Query System

System zapytań SQL-like dla TOON.

## Zawartość

- `demo.py` - System zapytań
- `run.sh` - Skrypt uruchamiający demo

## Uruchomienie

```bash
./run.sh
```

## Co pokazuje ten przykład

- Podstawowe zapytania SELECT
- Zapytania z JOIN (komendy i aliasy)
- Agregacje (GROUP BY, COUNT)
- Sortowanie i limitowanie wyników
