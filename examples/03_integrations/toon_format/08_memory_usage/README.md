# Demo 08: Memory Usage

Porównanie zużycia pamięci TOON vs stary system.

## Zawartość

- `demo.py` - Estymacja zużycia pamięci
- `run.sh` - Skrypt uruchamiający demo

## Uruchomienie

```bash
./run.sh
```

## Co pokazuje ten przykład

- Estymacja pamięci dla starego systemu (wiele plików)
- Estymacja pamięci dla TOON (zintegrowany plik)
- Porównanie rozmiarów
- Obliczanie oszczędności
- Wnioski o efektywności pamięciowej
