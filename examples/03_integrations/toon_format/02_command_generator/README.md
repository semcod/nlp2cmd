# Demo 02: Command Generator

Prosty generator komend w oparciu o TOON.

## Zawartość

- `demo.py` - Generator komend
- `run.sh` - Skrypt uruchamiający demo

## Uruchomienie

```bash
./run.sh
```

## Co pokazuje ten przykład

- Klasa `SimpleCommandGenerator`
- Metoda `get_command_info()` - pobieranie informacji o komendzie
- Metoda `list_by_category()` - listowanie komend z kategorii
