# Demo 11: Basic Integration

Podstawowa integracja z systemem TOON.

## Zawartość

- `demo.py` - Podstawowa integracja
- `run.sh` - Skrypt uruchamiający demo

## Uruchomienie

```bash
./run.sh
```

## Co pokazuje ten przykład

- Inicjalizacja menedżera danych TOON
- Pobieranie komend (shell, browser)
- Pobieranie konfiguracji
- Wyszukiwanie komend
