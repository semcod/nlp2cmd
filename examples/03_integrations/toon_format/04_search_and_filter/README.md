# Demo 04: Search and Filter

Wyszukiwanie i filtrowanie komend TOON.

## Zawartość

- `demo.py` - Wyszukiwanie i filtrowanie
- `run.sh` - Skrypt uruchamiający demo

## Uruchomienie

```bash
./run.sh
```

## Co pokazuje ten przykład

- Wyszukiwanie komend po nazwie
- Wyszukiwanie po opisie
- Filtrowanie po kategorii
- Filtrowanie po typie (shell, sql, kubernetes)
- Przykładowe wyniki wyszukiwania
