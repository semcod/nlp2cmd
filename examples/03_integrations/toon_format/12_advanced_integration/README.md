# Demo 12: Advanced Integration

Zaawansowana integracja z systemem TOON.

## Zawartość

- `demo.py` - Zaawansowana integracja
- `run.sh` - Skrypt uruchamiający demo

## Uruchomienie

```bash
./run.sh
```

## Co pokazuje ten przykład

- Cache'owanie wyników
- Wielowątkowy dostęp (thread-safe)
- Rozszerzone filtrowanie po tagach/złożoności
- Dynamiczne ładowanie kategorii
