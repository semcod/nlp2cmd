# Demo 06: Old System Mock

Mock starego systemu JSON/YAML dla porównania z TOON.

## Zawartość

- `demo.py` - Mock starego systemu
- `run.sh` - Skrypt uruchamiający demo

## Uruchomienie

```bash
./run.sh
```

## Co pokazuje ten przykład

- Klasa `OldSystemLoader` - mock starego systemu
- Ładowanie wielu plików JSON
- Cache'owanie wyników
- Wyszukiwanie komend po nazwie
