# Demo 10: Migration Guide

Przewodnik po migracji ze starego systemu do TOON.

## Zawartość

- `demo.py` - Przewodnik migracji
- `run.sh` - Skrypt uruchamiający demo

## Uruchomienie

```bash
./run.sh
```

## Co pokazuje ten przykład

- Struktura starego systemu (wiele plików)
- Struktura TOON (jeden plik)
- 4 kroki migracji:
  1. Zbierz wszystkie pliki JSON
  2. Utwórz strukturę TOON
  3. Konwersja danych
  4. Walidacja
- Przykład konwersji jednej komendy
