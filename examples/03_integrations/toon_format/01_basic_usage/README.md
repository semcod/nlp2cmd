# Demo 01: Basic TOON Usage

Podstawowe użycie formatu TOON (Type-Object-Open-Notation).

## Zawartość

- `demo.py` - Podstawowe użycie TOON
- `run.sh` - Skrypt uruchamiający demo

## Uruchomienie

```bash
./run.sh
```

## Co pokazuje ten przykład

- Inicjalizacja `get_data_manager()`
- Pobieranie komend różnych typów
- Przykładowe dane TOON dla shell, sql, kubernetes
