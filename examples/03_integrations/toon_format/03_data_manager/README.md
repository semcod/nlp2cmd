# Demo 03: Data Manager

Użycie Data Manager do zarządzania danymi TOON.

## Zawartość

- `demo.py` - Operacje Data Manager
- `run.sh` - Skrypt uruchamiający demo

## Uruchomienie

```bash
./run.sh
```

## Co pokazuje ten przykład

- Pobieranie wszystkich komend
- Filtrowanie po typie (shell, sql)
- Pobieranie komendy po nazwie
- Pobieranie konfiguracji i ustawień LLM
- Struktury danych TOON
