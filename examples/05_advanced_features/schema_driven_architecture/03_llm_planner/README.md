# Demo 03: LLM Planner

Demonstracja LLM Planner - generowanie wieloetapowego planu.

## Zawartość

- `demo.py` - LLM Planner
- `run.sh` - Skrypt uruchamiający demo

## Uruchomienie

```bash
./run.sh
```

## Co pokazuje ten przykład

- Klasa `PlanStep` - pojedynczy krok planu
- Klasa `LLMPlanner` - generowanie planów
- Przykładowe plany (backup, deploy, logs)
- Kiedy używać LLM Planner
