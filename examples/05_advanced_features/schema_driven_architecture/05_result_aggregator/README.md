# Demo 05: Result Aggregator

Demonstracja Result Aggregator - formatowanie i agregacja wyników.

## Zawartość

- `demo.py` - Result Aggregator
- `run.sh` - Skrypt uruchamiający demo

## Uruchomienie

```bash
./run.sh
```

## Co pokazuje ten przykład

- Enum `OutputFormat` (JSON, YAML, TABLE, MARKDOWN)
- Klasa `ExecutionResult`
- Klasa `ResultAggregator`
- Konwersja wyników do różnych formatów
- Zastosowania agregacji wyników
