# Demo 04: Plan Executor

Demonstracja Plan Executor - wykonanie typowanych akcji.

## Zawartość

- `demo.py` - Plan Executor
- `run.sh` - Skrypt uruchamiający demo

## Uruchomienie

```bash
./run.sh
```

- Klasa `ExecutionResult` - wynik wykonania
- Klasa `PlanExecutor` - wykonawca planu
- Handlery akcji (mock)
- Wykonywanie kroków planu
- Obsługa błędów
