# 04 — Browser Tabs: Multi-Tab Management

Open Firefox and manage multiple browser tabs via nlp2cmd.

## What it does

1. Opens Firefox browser
2. Navigates to GitHub
3. Opens a new tab with Stack Overflow
4. Opens a third tab with DuckDuckGo
5. Switches between tabs
6. Closes one tab

## Run

```bash
cd examples/06_desktop_automation/04_browser_tabs
bash run.sh
```

## Output

After running, check `logs/session.md` for the Markdown log with screenshots.
