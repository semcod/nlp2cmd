# 05 — Email Client: Thunderbird Automation

Control Thunderbird email client via nlp2cmd — check mail, compose messages.

## What it does

1. Launches Thunderbird
2. Checks for new messages (Ctrl+Shift+T)
3. Composes a new email (Ctrl+N)
4. Types recipient, subject, and body
5. Takes screenshot of the composed email

## Run

```bash
cd examples/06_desktop_automation/05_email_client
bash run.sh
```

## Output

After running, check `logs/session.md` for the Markdown log with screenshots.
