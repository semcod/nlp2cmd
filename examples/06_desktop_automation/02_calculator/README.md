# 02 — Calculator: Do Math

Open galculator and perform calculations on the remote desktop.

## What it does

1. Opens terminal (Ctrl+Alt+T)
2. Launches `galculator` from terminal
3. Types `42*137` and presses Enter (expected: 5754)
4. Types `256*256` and presses Enter (expected: 65536)

## Run

```bash
cd examples/06_desktop_automation/02_calculator
bash run.sh
```

## Output

After running, check `logs/session.md` for the Markdown log with screenshots.
