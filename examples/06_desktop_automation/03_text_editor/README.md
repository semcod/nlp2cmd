# 03 — Text Editor: Write and Save a Document

Open Mousepad text editor, write a document, and save it.

## What it does

1. Opens Application Finder (Alt+F2)
2. Launches Mousepad
3. Types a document with title, date, and content
4. Saves with Ctrl+S to `/home/nlp2cmd/report.txt`

## Run

```bash
cd examples/06_desktop_automation/03_text_editor
bash run.sh
```

## Output

After running, check `logs/session.md` for the Markdown log with screenshots.
