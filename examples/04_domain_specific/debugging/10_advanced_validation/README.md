# Demo 10: Advanced Validation

Zaawansowane techniki walidacji komend.

## Zawartość

- `demo.py` - Zaawansowana walidacja
- `run.sh` - Skrypt uruchamiający demo

## Uruchomienie

```bash
./run.sh
```

## Co pokazuje ten przykład

- Klasa `ValidationResult` - wynik walidacji
- Klasa `AdvancedValidator` - zaawansowany walidator
- Metryka podobieństwa komend
- Podsumowanie wyników walidacji
