# Demo 05: Real World Examples

Demonstracja rzeczywistych przypadków użycia nlp2cmd.

## Zawartość

- `demo.py` - Rzeczywiste przypadki użycia
- `run.sh` - Skrypt uruchamiający demo

## Uruchomienie

```bash
./run.sh
```

## Co pokazuje ten przykład

Trzy rzeczywiste scenariusze:

1. **DevOps Automation**
   - Monitorowanie i optymalizacja
   - Deploy aplikacji

2. **Data Science**
   - Analiza danych
   - Wizualizacja

3. **System Administration**
   - Zarządzanie systemem
   - Czyszczenie logów
   - Sprawdzanie bezpieczeństwa
