# Demo 09: Network Commands Tests

Testy dla komend sieciowych.

## Zawartość

- `demo.py` - Testy komend sieciowych
- `run.sh` - Skrypt uruchamiający demo

## Uruchomienie

```bash
./run.sh
```

## Co pokazuje ten przykład

- Testy ping
- Testy konfiguracji IP
- Testy otwartych portów
- Testy pobierania plików (curl)
- Testy DNS (nslookup)
