# Demo 06: Test Framework

Podstawy frameworka testowego dla komend shell.

## Zawartość

- `demo.py` - Podstawy test framework
- `run.sh` - Skrypt uruchamiający demo

## Uruchomienie

```bash
./run.sh
```

## Co pokazuje ten przykład

- Klasa `CommandTest` - definicja testu
- Klasa `SimpleTestRunner` - runner testów
- Dodawanie testów
- Uruchamianie testów
- Podsumowanie wyników
