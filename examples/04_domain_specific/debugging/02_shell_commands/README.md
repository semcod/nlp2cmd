# Demo 02: Shell Commands

Demonstracja komend shell nlp2cmd.

## Zawartość

- `demo.py` - Przykłady komend shell
- `run.sh` - Skrypt uruchamiający demo

## Uruchomienie

```bash
./run.sh
```

## Co pokazuje ten przykład

- Podstawowe komendy nlp2cmd
- Różne DSL (shell, sql, docker, kubernetes)
- Opcje: --explain, --auto-repair, --interactive
- Analiza środowiska
- Walidacja i naprawa
