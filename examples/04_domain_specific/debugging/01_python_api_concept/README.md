# Demo 01: Python API Concept

Demonstracja koncepcji użycia nlp2cmd przez Python API.

## Zawartość

- `demo.py` - Przykład użycia Python API
- `run.sh` - Skrypt uruchamiający demo

## Uruchomienie

```bash
./run.sh
```

Lub bezpośrednio:

```bash
python3 demo.py
```

## Co pokazuje ten przykład

- Import i inicjalizację `HybridThermodynamicGenerator`
- Przykładowe zapytania DSL
- Przykładowe zapytania thermodynamic
- Typy wyników dla różnych zapytań
