# Demo 04: Advanced Patterns

Demonstracja zaawansowanych wzorców użycia nlp2cmd.

## Zawartość

- `demo.py` - Zaawansowane wzorce
- `run.sh` - Skrypt uruchamiający demo

## Uruchomienie

```bash
./run.sh
```

## Co pokazuje ten przykład

- Batch Processing (Python) - przetwarzanie wielu zapytań równolegle
- Pipeline (Shell) - łączenie komend w potok
- Z pliku (Shell) - wczytywanie zapytań z pliku
- Kontekstowe zapytania (Python) - zapytania z dodatkowym kontekstem
