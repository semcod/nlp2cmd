# Demo 08: System Commands Tests

Testy dla komend systemowych.

## Zawartość

- `demo.py` - Testy komend systemowych
- `run.sh` - Skrypt uruchamiający demo

## Uruchomienie

```bash
./run.sh
```

## Co pokazuje ten przykład

- Testy listy użytkowników
- Testy zużycia pamięci
- Testy procesów CPU
- Testy miejsca na dysku
- Testy uptime systemu
