# Demo 03: Mixed Workflow

Demonstracja mieszanego workflow Python + Shell.

## Zawartość

- `demo.py` - Przykład mieszanego workflow
- `run.sh` - Skrypt uruchamiający demo

## Uruchomienie

```bash
./run.sh
```

## Co pokazuje ten przykład

- Scenariusz optymalizacji systemu
- Workflow 4-krokowy:
  1. Analiza środowiska (shell)
  2. Generowanie rozwiązań (Python)
  3. Wykonanie komend (shell)
  4. Walidacja wyników (shell)
