# Demo 01: PDF Extraction

Ekstrakcja tekstu z plików PDF.

## Zawartość

- `demo.py` - Ekstrakcja tekstu z PDF
- `run.sh` - Skrypt uruchamiający demo

## Uruchomienie

```bash
./run.sh
```

## Co pokazuje ten przykład

- Klasa `PDFExtractor` - ekstraktor tekstu
- Ekstrakcja tekstu z pojedynczego PDF
- Ekstrakcja metadanych (tytuł, strony, autor)
- Batch processing wielu plików
- Cache'owanie wyników
