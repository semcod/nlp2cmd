# Demo 03: LLM Search

Wyszukiwanie informacji w PDF za pomocą LLM.

## Zawartość

- `demo.py` - LLM Search
- `run.sh` - Skrypt uruchamiający demo

## Uruchomienie

```bash
./run.sh
```

## Co pokazuje ten przykład

- Klasa `LLMSearcher` - wyszukiwanie informacji
- Obliczanie relevance score
- Sortowanie wyników po relevance
- Ekstrakcja pasujących fragmentów
- Historia zapytań
