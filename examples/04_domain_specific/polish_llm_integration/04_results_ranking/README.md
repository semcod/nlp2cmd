# Demo 04: Results Ranking

Ranking i filtrowanie wyników wyszukiwania.

## Zawartość

- `demo.py` - Results Ranking
- `run.sh` - Skrypt uruchamiający demo

## Uruchomienie

```bash
./run.sh
```

## Co pokazuje ten przykład

- Klasa `SearchResult` - struktura wyniku
- Klasa `ResultsRanker` - ranking wyników
- Filtrowanie po min relevance
- Sortowanie po relevance
- Dywersyfikacja źródeł
- Top K wyników
