# Demo 02: File Processing

Przetwarzanie plików bioinformatycznych.

## Zawartość

- `demo.py` - Przetwarzanie plików
- `run.sh` - Skrypt uruchamiający demo

## Uruchomienie

```bash
./run.sh
```

## Co pokazuje ten przykład

- Wypisywanie nagłówków FASTA
- Filtrowanie sekwencji po długości
- Dzielenie plików na części
- Usuwanie duplikatów
- Łączenie plików FASTA
