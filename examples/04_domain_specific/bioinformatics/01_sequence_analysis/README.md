# Demo 01: Sequence Analysis

Analiza sekwencji biologicznych.

## Zawartość

- `demo.py` - Analiza sekwencji
- `run.sh` - Skrypt uruchamiający demo

## Uruchomienie

```bash
./run.sh
```

## Co pokazuje ten przykład

- Wyszukiwanie sekwencji FASTA
- Liczenie długości sekwencji
- Konwersja formatów (FASTA → tabular)
- Wyszukiwanie motywów DNA
- Odwracanie i uzupełnianie sekwencji
