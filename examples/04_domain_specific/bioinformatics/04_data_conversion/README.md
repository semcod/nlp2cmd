# Demo 04: Data Conversion

Konwersja formatów danych bioinformatycznych.

## Zawartość

- `demo.py` - Konwersja formatów
- `run.sh` - Skrypt uruchamiający demo

## Uruchomienie

```bash
./run.sh
```

## Co pokazuje ten przykład

- Konwersja FASTA do FASTQ
- Konwersja VCF do TSV
- Konwersja BAM do SAM
- Konwersja GFF3 do BED
- Konwersja DNA do RNA
