# Demo 03: BLAST Operations

Operacje wyszukiwania BLAST.

## Zawartość

- `demo.py` - Operacje BLAST
- `run.sh` - Skrypt uruchamiający demo

## Uruchomienie

```bash
./run.sh
```

## Co pokazuje ten przykład

- Tworzenie bazy danych BLAST
- Wyszukiwanie sekwencji nucleotide (blastn)
- Wyszukiwanie białek (blastp)
- Filtrowanie po e-value
- Ograniczanie liczby wyników
